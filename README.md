# incc-lsp-vscode-extension


# Install
```sh
sh install_extension.sh
```
# Uninstall
```sh
sh uninstall_extension.sh
```

# Restart Server:
- open commandPalette
`shift+cmd+p`

- restart:
```
lsp-incc-vscode.restart
```
